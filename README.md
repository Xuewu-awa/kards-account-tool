## 功能说明

一键创建Kards游戏设备账号并绑定邮箱。

### 建号流程说明

脚本自动完成以下步骤：

1. 生成设备ID和密码
2. 模拟设备信息发送给服务器
3. 自动创建游戏账号
4. 设置玩家名称
5. 跳过所有新手教程
6. 领取所有可用兑换码
7. 将邮箱绑定到该设备账号

## 使用方法

```bash
# 安装依赖
pip install requests

# 运行脚本
python main.py
```

## 按提示操作：

1. 输入玩家名称
2. 输入要绑定的邮箱和密码
3. 账号信息自动保存

## 文件说明

· accounts.txt - 已绑定邮箱的账号，格式：邮箱|邮箱密码|设备用户名|设备密码|显示名称
· device_accounts.txt - 绑定失败的设备账号（可选保存）
